# 💰 Finance Dashboard

This is a simple and interactive **Finance Dashboard UI** built using HTML, CSS, and JavaScript.

## 🚀 Features

- 📊 Dashboard summary (Balance, Income, Expense)
- ➕ Add / Edit / Delete transactions
- 🔐 Role-based UI (Admin / Viewer)
- 🔍 Search and filter transactions
- 💡 Insights (Highest spending category)
- 💾 Data persistence using LocalStorage
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 🌐 Live Demo

👉 https://rashmihaudhary.github.io/Finance-dashboard/

## 📂 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser

This project was created as part of an assignment to demonstrate frontend development skills including UI design, state management, and user interaction.
